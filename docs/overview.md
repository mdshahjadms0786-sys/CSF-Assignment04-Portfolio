# Project Overview

## 1. Purpose
The purpose of this project is to create a clean, simple, and responsive portfolio for the **Computer Science Fundamentals Assignment**.  
This portfolio represents my understanding of HTML, CSS, folder structure, and version control using Git & GitHub.

---

## 2. Project Features
### ✔ About Section  
Small introduction about myself with a profile image.

### ✔ Skills Section  
List of technical skills with simple styling.

### ✔ Projects Section  
Shows my completed mini-projects with descriptions.

### ✔ Contact Section  
Email + GitHub link.

---

## 3. Tech Stack
- **HTML5** – For structure  
- **CSS3** – For styling + responsiveness  
- **Git** – For version control  
- **GitHub** – For hosting code online
